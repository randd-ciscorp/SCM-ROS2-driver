# SCM ROS 2 driver

ROS 2 driver node for CIS SCM camera series.

## Required system configurations

- Host Linux PC/Robot
    - Supported OS / ROS distro
        - Ubuntu 22.04 / ROS 2 `Humble`
        - Ubuntu 24.04 / ROS 2 `Jazzy`, `Kilted`, `Rolling`
    - USB3.0 port

- SCM camera
- USB3.0 / micro USB type-B cable
- Power supply (12V 5A)
